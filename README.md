# PCOS Detection and Clinical Insight Analysis

This project focuses on predicting Polycystic Ovary Syndrome (PCOS) using Machine Learning techniques and analyzing key health indicators.

## Project Overview
- Analyzed dataset with 500+ patient records and multiple health features
- Performed data preprocessing and cleaning
- Conducted Exploratory Data Analysis (EDA)

## 🤖 Machine Learning Models
- Logistic Regression
- Random Forest Classifier

## Techniques Used
- Classification for PCOS prediction
- K-Means Clustering for grouping patients
- Data Visualization using Matplotlib and Seaborn

## Results
- Achieved ~73% accuracy
- Identified important health patterns

##  Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
- Jupyter Notebook
